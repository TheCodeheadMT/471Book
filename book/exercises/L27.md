**1. Logistic Regression Prediction**
A logistic regression model is trained to predict whether a drone's battery will fail during a sortie. The model uses a single feature: Battery Temperature in Celsius ($x$). The trained parameters are an intercept $\beta_0 = -6.5$ and a weight $\beta_1 = 0.15$. If a drone reports a battery temperature of $40^\circ\text{C}$, what is the probability of failure? Does the model predict failure (assuming a 0.5 threshold)?

<details>
  <summary>Show solution</summary>
<ul>
  <li><strong>Step 1:</strong> Calculate the linear combination <i>z</i> = &beta;<sub>0</sub> + &beta;<sub>1</sub><i>x</i>.<br>
  <i>z</i> = -6.5 + (0.15 &times; 40) = -6.5 + 6.0 = -0.5</li>
  <li><strong>Step 2:</strong> Apply the sigmoid function <i>P</i> = 1 / (1 + <i>e</i><sup>-<i>z</i></sup>).<br>
  <i>P</i> = 1 / (1 + <i>e</i><sup>-(-0.5)</sup>) = 1 / (1 + <i>e</i><sup>0.5</sup>)</li>
  <li><strong>Step 3:</strong> Calculate the final probability (using <i>e</i><sup>0.5</sup> &approx; 1.649).<br>
  <i>P</i> = 1 / (1 + 1.649) = 1 / 2.649 &approx; 0.377</li>
  <li><strong>Answer:</strong> The probability of failure is <strong>37.7%</strong>. Since 0.377 &lt; 0.5, the model predicts the battery will <strong>not</strong> fail.</li>
</ul>
</details>

**2. Linear Discriminant Analysis (Bayes Theorem)**
An LDA model classifies incoming aircraft as Friendly or Unknown. Historically, 80% of aircraft in this sector are Friendly ($\pi_F = 0.8$) and 20% are Unknown ($\pi_U = 0.2$). Radar picks up an aircraft moving at Mach 1.2. The probability density function evaluates this speed for Friendly aircraft as $f_F(\text{Mach } 1.2) = 0.05$, and for Unknown aircraft as $f_U(\text{Mach } 1.2) = 0.30$. Calculate the probability that the aircraft is Unknown.

<details>
  <summary>Show solution</summary>
<ul>
  <li><strong>Step 1:</strong> Calculate the Unknown score (Numerator): <i>f</i><sub>U</sub>(<i>x</i>)&pi;<sub>U</sub>.<br>
  0.30 &times; 0.2 = 0.06</li>
  <li><strong>Step 2:</strong> Calculate the Friendly score: <i>f</i><sub>F</sub>(<i>x</i>)&pi;<sub>F</sub>.<br>
  0.05 &times; 0.8 = 0.04</li>
  <li><strong>Step 3:</strong> Calculate the total probability of seeing this evidence (Denominator).<br>
  0.06 + 0.04 = 0.10</li>
  <li><strong>Step 4:</strong> Divide the Unknown score by the total.<br>
  <i>P</i>(Unknown|Mach 1.2) = 0.06 / 0.10 = 0.60</li>
  <li><strong>Answer:</strong> There is a <strong>60%</strong> probability the aircraft is Unknown.</li>
</ul>
</details>

**3. SVM Margin Calculation**
A Support Vector Machine has found the optimal linear hyperplane separating stealth and non-stealth radar signatures. The weight vector for this boundary is $w = [6, 8]$. Calculate the width of the margin separating the two classes.

<details>
  <summary>Show solution</summary>
<ul>
  <li><strong>Step 1:</strong> Calculate the magnitude (Euclidean norm) of the weight vector ||<i>w</i>|| = &radic;(<i>w</i><sub>1</sub><sup>2</sup> + <i>w</i><sub>2</sub><sup>2</sup>).<br>
  ||<i>w</i>|| = &radic;(6<sup>2</sup> + 8<sup>2</sup>) = &radic;(36 + 64) = &radic;(100) = 10</li>
  <li><strong>Step 2:</strong> Apply the margin formula Margin = 2 / ||<i>w</i>||.<br>
  Margin = 2 / 10 = 0.2</li>
  <li><strong>Answer:</strong> The margin width is <strong>0.2</strong>.</li>
</ul>
</details>

**4. Bias-Variance Tradeoff**
An AI team evaluates a new neural network for target recognition. They determine the model's squared bias is $0.04$, its variance is $0.08$, and the inherent sensor noise (irreducible error) is $0.02$. What is the total expected error of the model? If they switch to a simpler linear model that halves the variance but doubles the squared bias, what is the new expected error?
<details>
  <summary>Show solution</summary>
<ul>
  <li><strong>Step 1:</strong> Calculate total error for the neural network: Error = Bias<sup>2</sup> + Variance + Noise.<br>
  Error<sub>NN</sub> = 0.04 + 0.08 + 0.02 = 0.14 (or 14%)</li>
  <li><strong>Step 2:</strong> Calculate the parameters for the linear model.<br>
  New Bias<sup>2</sup> = 0.04 &times; 2 = 0.08<br>
  New Variance = 0.08 / 2 = 0.04</li>
  <li><strong>Step 3:</strong> Calculate total error for the linear model.<br>
  Error<sub>Linear</sub> = 0.08 + 0.04 + 0.02 = 0.14</li>
  <li><strong>Answer:</strong> Both models have an expected error of <strong>14%</strong>. This illustrates that simply moving along the tradeoff curve doesn't inherently reduce total error unless you find a better model architecture.</li>
</ul>
</details>

**5. Decision Tree Gini Impurity**
A decision tree node evaluates a group of 20 past sorties. 16 of the sorties were "Success" and 4 were "Failure". Calculate the Gini Impurity of this node.


<details>
  <summary>Show solution</summary>
<ul>
  <li><strong>Step 1:</strong> Calculate the probability of each class (<i>p<sub>i</sub></i>).<br>
  <i>p</i><sub>Success</sub> = 16 / 20 = 0.8<br>
  <i>p</i><sub>Failure</sub> = 4 / 20 = 0.2</li>
  <li><strong>Step 2:</strong> Square the probabilities.<br>
  0.8<sup>2</sup> = 0.64<br>
  0.2<sup>2</sup> = 0.04</li>
  <li><strong>Step 3:</strong> Apply the Gini formula Gini = 1 - &sum;<i>p<sub>i</sub></i><sup>2</sup>.<br>
  Gini = 1 - (0.64 + 0.04) = 1 - 0.68 = 0.32</li>
  <li><strong>Answer:</strong> The Gini Impurity of the node is <strong>0.32</strong>.</li>
</ul>
</details>

**6. Random Forest Variance Reduction**
A single, highly overfitted decision tree has a prediction variance of $0.50$. You build a Random Forest by bagging 10 of these trees. Assuming the trees are perfectly independent (an idealized mathematical scenario), what is the variance of the resulting Random Forest?

<details>
  <summary>Show solution</summary>
<ul>
  <li><strong>Step 1:</strong> Use the statistical property that the variance of an average of <i>N</i> independent variables is Variance / <i>N</i>.</li>
  <li><strong>Step 2:</strong> Divide the single tree variance by the number of trees.<br>
  Variance<sub>RF</sub> = 0.50 / 10 = 0.05</li>
  <li><strong>Answer:</strong> The idealized variance drops to <strong>0.05</strong>. (Note: In reality, because the trees train on overlapping subsets of the same data, they are correlated, so the variance reduction will be slightly less perfect, but still highly effective).</li>
</ul>
</details>
